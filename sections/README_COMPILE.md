# Compiling `PHASE4_FORMAL_REPORT_MASTER.tex` to PDF

The master report assembles three modular, context-window-friendly sections
via `\input{}`:

```
PHASE4_FORMAL_REPORT_MASTER.tex
├── sections/sec_lean4_proofs.tex          (mirrored by .md)
├── sections/sec_python_visualization.tex  (mirrored by .md)
└── sections/sec_context_explanation.tex   (mirrored by .md)
```

Figures are read from `figures/*.png`, generated by `figures/generate_figures.py`.

## Prerequisite: LaTeX Distribution

**No LaTeX engine (`pdflatex`/`xelatex`/`tectonic`) was found on this machine.**
To compile the combined PDF, install one of:

- **MiKTeX** (Windows, recommended): https://miktex.org/download
- **TeX Live**: https://tug.org/texlive/
- **Tectonic** (lightweight, self-contained): `winget install tectonic-typesetting.tectonic`

## Compile Commands

Once a LaTeX distribution is installed, run from the repository root:

```powershell
# Regenerate figures (if discoveries data changed)
python figures/generate_figures.py

# Compile (run twice for table of contents / references)
pdflatex -interaction=nonstopmode PHASE4_FORMAL_REPORT_MASTER.tex
pdflatex -interaction=nonstopmode PHASE4_FORMAL_REPORT_MASTER.tex
```

Or with Tectonic (single pass, no separate TOC re-run needed):

```powershell
tectonic PHASE4_FORMAL_REPORT_MASTER.tex
```

Output: `PHASE4_FORMAL_REPORT_MASTER.pdf`

## Reviewing Sections Independently (Context-Window Management)

Each `.md` file under `sections/` can be reviewed standalone by an LLM or
human reviewer without loading the full combined document:

- `sections/sec_lean4_proofs.md` — Lean 4 formal proofs (S11 falsification, S12/S21 K3 validation, Chameleon stability, PAC bounds)
- `sections/sec_python_visualization.md` — Figure descriptions and reproducibility
- `sections/sec_context_explanation.md` — Combined proof narrative and astrophysical bridge

The `.tex` counterparts contain the same content with LaTeX-specific
formatting (equations, `lstlisting` code blocks, `\input` figure references).
